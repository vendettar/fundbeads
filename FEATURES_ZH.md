# Fundbeads 功能

本文只描述当前已实现的功能。计划项尚未实现。

## 当前

- 通过浏览器文件选择器本地上传 JPG/PNG 图片。
- 在浏览器本地完成图片解码、canvas 采样、色板匹配和计数。
- 输出尺寸：`52x52`、`64x64`、`78x78`。
- 内置 28 色模拟 MARD 色板。
- 使用平方 RGB 欧氏距离做最近色匹配。
- 透明 PNG 像素会先按白色背景合成，再进行颜色匹配。
- 图纸网格中每个可见格代表一颗珠子。
- 每个珠子格内显示 MARD code。
- 顶部、底部、左侧、右侧都有编号坐标轴。
- 每 5 和 10 格显示更明显的计数辅助线。
- 图纸默认适配当前可视区域。
- 提供放大和缩小按钮，用于查看密集网格细节。
- 鼠标位于图纸区域时，支持 Ctrl + 鼠标滚轮缩放；macOS 支持 Command + 鼠标滚轮缩放。
- 颜色用量统计包含色块、MARD code、名称、数量、已用颜色数、图纸尺寸和总珠子数。
- 支持静态构建和 nginx Docker 运行方式。

## 当前限制

- 当前色板是模拟子集，不是完整 MARD 221 色数据集。
- 源图片会拉伸到正方形网格，尚未提供裁剪或适配控制。
- 尚未实现可打印导出。
- 尚未实现色板过滤。
- 很小的屏幕上可能需要放大后才能阅读单个珠子 code。

## 计划

- 接入经过验证的完整 MARD 221 色色板。
- 增加浏览器本地的可打印图纸导出。
- 增加可选色板过滤。
- 改进移动端网格浏览和触摸缩放体验。
