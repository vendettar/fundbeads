# Fundbeads 功能

本文只描述当前已实现的功能。计划项尚未实现。

## 当前

- 通过浏览器文件选择器本地上传 JPG/PNG 图片。
- 在浏览器本地完成图片解码、canvas 采样、色板匹配和计数。
- 最长边预设：`52`、`64`、`78`，并支持最长边在 `40` 到 `100` 之间调整。
- 输出尺寸会保持源图片比例，例如 `16:9` 图片在最长边 `80` 时生成 `80x45`。
- 内置静态 MARD 221 色板。
- 使用平方 RGB 欧氏距离做最近色匹配。
- 透明 PNG 像素会先按白色背景合成，再进行颜色匹配。
- 图纸网格中每个可见格代表一颗珠子。
- 每个珠子格内显示 MARD code。
- 顶部、底部、左侧、右侧都有编号坐标轴。
- 每 5 和 10 格显示更明显的计数辅助线。
- 图纸默认适配当前可视区域。
- 提供放大和缩小按钮，用于查看密集网格细节。
- 鼠标位于图纸区域时，支持 Ctrl + 鼠标滚轮缩放；macOS 支持 Command + 鼠标滚轮缩放。
- 颜色用量统计包含色块、MARD code、名称、数量、已用颜色数、图纸尺寸和总珠子数。
- 支持界面语言选择：英文、简体中文、繁体中文、日语、韩语、西班牙语。
- 界面文案已本地化，色板默认使用稳定的 MARD code 标签，并支持可选展示名覆盖。
- 支持主题选择：经典、午夜、海洋、糖果、单色。
- 支持界面风格选择：现代、像素。
- 语言、主题和界面风格偏好仅在可用时保存到浏览器 `localStorage`。
- 支持静态构建和 nginx Docker 运行方式。

## 当前限制

- 色板匹配使用内置 MARD 221 RGB 数据集。
- 输出尺寸会根据源图片比例和最长边自动计算，尚未提供固定尺寸裁剪或拖拽取景。
- 尚未实现可打印导出。
- 尚未实现色板过滤。
- 很小的屏幕上可能需要放大后才能阅读单个珠子 code。
- 翻译内容维护在静态源码中，不使用远程翻译服务。

## 计划

- 增加浏览器本地的可打印图纸导出。
- 增加可选色板过滤。
- 改进移动端网格浏览和触摸缩放体验。
